# Customer Churn Analysis

## Objective

The objective of this project is to analyze customer data to identify patterns behind subscription cancellations. The analysis evaluates customer activity, engagement levels, and behavioral trends to determine the key factors that lead to customer churn and suggest strategies for improving customer retention.

## Dataset

Telco Customer Churn Dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Workflow

1. Imported required libraries.
2. Loaded the customer churn dataset.
3. Explored the dataset structure.
4. Checked for missing values.
5. Removed duplicate records.
6. Cleaned the TotalCharges column.
7. Analyzed customer churn distribution.
8. Compared churn across contract types.
9. Analyzed monthly charges and customer tenure.
10. Examined internet service and payment methods.
11. Exported the cleaned dataset.

## Output Files

- customer_churn_analysis.ipynb
- cleaned_customer_churn.csv

## Conclusion

The analysis shows that customers with month-to-month contracts, shorter tenure, higher monthly charges, and certain payment methods are more likely to churn. These insights can help businesses design effective customer retention strategies.
